The first plan

This week my main goal is to get up and running so that I can be more productive in the future. I need to finish getting this website running and start to publish the content online. Should I managed to get this done I will be able to move onto my first experiments of the PhD (Yay).

  1. Write the week's plan
  2. Build the Website
  3. Publish the aforementioned plan

Bonus Objectives:

At the end of each week I will refer back to this plan to see how successful or not I have been. I will likely put a few blog posts up in between but this is a good starting schedule!
